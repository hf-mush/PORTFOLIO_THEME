# PORTFOLIO_THEME

PORTFOLIO_THEME is wordpress themes.

## Environment 

* MacOS 10.14.6 Mojave
* zsh 5.3 (x86_64-apple-darwin18.0)
* Node v10.0.0
* npm v6.9.0
* PHP 7.3.0
* Wordpress 5.2.4

## Deploy

### .env

```
$ cd cms
$ cp .env.local .env
$ vim .env
...
```

### upload shell

```
$ sh cms/upload
>> cms upload success.
```
